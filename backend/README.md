# Blog API

These docs are for Jacob to get him up to speed.

## Usage

To start the server run:

```
uvicorn blog_api:app --reload
```

## Setup

```bash
cd gt-backend
python -m venv venv
./venv/Scripts/activate.ps1 (or .bat, if you aren't using powershell)
pip install -r requirements.txt
python -m blog_api
```
